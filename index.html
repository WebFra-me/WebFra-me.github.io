Features

WebFrame is feature packed to facilitate the simple and easy creation, integration, and usage of web applications. We are also working constantly to expand WebFrame's feature set. Here are some of the notable features:

  • Versitile framework for the easy creation of web applications
  • Authentication and security built-in
  • Customizable layouts and themes to fit any need
  • Tools to assist in the development of web applications
  • Marketplace with many web applications developed by other WebFrame users
  • Completely open source (See how you can get involved)

Get Involved

WebFrame is completely open source, which means we rely on user support for the advancement in development of WebFrame. There are a few ways you can get ivolved to help improve WebFrame for everyone!
  • Develop for WebFrame
  • Contribute to our documentation
  • Build web applications and publish them on the Marketplace
  • Provide feedback regarding issues, improvements, or requests for new features
  • Spread the word!


WebFrame is hosted on GitHub. To contribute to WebFrame development or documentation, please head to GitHub.
